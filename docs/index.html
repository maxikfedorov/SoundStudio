Студия звукозаписи "Sound Studio"

Веб-приложение на React

В моей студии работает персонал из нескольких человек, включая меня. Мы можем предоставлять услуги звукорежиссуры (запись, сведение, мастеринг, продакшн и т.д), а так же предоставлять список курсов по звуку (видеоролики интенсивов по сведению, мастерингу, записи и т.д.). Мы продаём свои услуги через сайт, где пользователь, зарегистрировавшись, добавляет в корзину понравившиеся ему товары (услуги/курсы), после чего оплачивает.
